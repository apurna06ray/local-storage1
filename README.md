#  Quick Notes Chrome Extension

A simple and useful Chrome extension to quickly save notes directly from your browser.

---

##  Features

*  Write notes instantly
*  Save notes in browser storage
*  Auto-load saved notes
*  Clean and simple UI

---

##  How to Use

1. Download this repository as ZIP
2. Extract the ZIP file
3. Open Chrome and go to: chrome://extensions/
4. Enable **Developer Mode**
5. Click **Load Unpacked**
6. Select the extracted folder

---

##  Demo

Open the extension from the toolbar, write a note, and click **Save**.
Your note will be stored and appear again when you reopen the extension.
